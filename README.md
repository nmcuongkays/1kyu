# Kinoukentei Machine 1kyu PWA

Bản app ôn nhận diện hư hỏng dao cắt cho 技能検定 機械加工 1級.

## Cách dùng nhanh
1. Mở `index.html` trên máy tính để test.
2. Để dùng như app trên iPhone:
   - Upload nguyên thư mục này lên GitHub Pages hoặc Netlify.
   - Mở link bằng Safari.
   - Bấm Share → Add to Home Screen.
3. App tự lưu tiến độ/câu sai bằng localStorage.

## Chức năng
- Thẻ học từng hiện tượng.
- Quiz nhìn hình chọn tên.
- Quiz đọc định nghĩa.
- Quiz đọc đối sách.
- Đề mô phỏng.
- Tự lưu câu sai.
- Thống kê theo nhóm kiến thức.
- Ảnh minh họa cắt từ ảnh đề đã gửi.

## Sửa dữ liệu
Mở `app.js`, phần `window.KINOUKENTEI_DATA`.
